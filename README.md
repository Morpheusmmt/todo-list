# 📋 Task Manager  

Um gerenciador de tarefas simples em **JavaScript puro**, HTML e CSS.  

## 🚀 Funcionalidades  
- **Adicionar tarefas** (título + descrição)  
- **Excluir tarefas**  
- **Marcar como concluídas** ou **reabri-las**  
- **Filtro** para visualizar:  
  - Todas as tarefas  
  - Ativas  
  - Concluídas  

## 🛠️ Tecnologias  
- HTML5  
- CSS3  
- JavaScript (Vanilla)
